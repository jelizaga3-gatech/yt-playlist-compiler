Youtube Video Uploader

What is this?

This simple tool lets you create a YouTube playlist from a list of videos.

This application requires access to a Google account.

This tool can be interfaced with applications that generate YouTube playlists. Send me a recommendation here.

Instructions

  1. Log in.
  2. View or create a playlist.
  3. Select a playlist to upload to.
  4. Add videos and upload.
  5. Provide feedback or say thanks, if it helped. :-)
