Youtube Video Id Extractor

Description

This simple tool lets you create a YouTube playlist from a Udacity course.

{#

Note: This application does store cookies. Using this application Use at your own risk.

#}

Privacy Policy: You **must** log-in using Udacity's sign-in page to view your available courses. A 1 minute timeout is set during this time frame. You will also be asked to allow YouTube to authorize this application via the OAuth 2.0 authorization process.

Cookies are stored for authentication. Use at your own risk.

Instructions

  1. Click the Get Courses button. This will open a pop up to Udacity's Log In page. You will have 60 seconds to log in, otherwise you may have to start over. Unfortunately, Udacity does not provide an easy way to access its resources.
  2. A list of available courses to select from will be shown below "Course List".
  3. Please select a course and click "Get Lesson Links".
  4. A list of available lessons will be shown below "Lesson Links".
  5. Please select a lesson and click "Get YouTube Links".
